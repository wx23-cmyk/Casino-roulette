# Casino Roulette

## Project Overview
Casino Roulette is an interactive game simulation that allows users to play roulette online. The project employs various programming techniques to create a realistic gaming experience. The game features a user interface that simulates a physical roulette table, and users can place bets on different outcomes.

## Features
- User-friendly interface for easy navigation.
- Realistic simulation of roulette mechanics.
- Support for multiple betting options (e.g., single numbers, colors, odd/even).
- Recording and display of user game history.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/wx23-cmyk/Casino-roulette.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Casino-roulette
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## How to Play
1. Start the game:
   ```bash
   npm start
   ```
2. Place your bets by selecting your desired numbers or colors.
3. Spin the wheel to see the outcome.
4. Track your winnings and losses in your game history.
